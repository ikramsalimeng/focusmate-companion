# Security Policy

## Reporting a vulnerability

If you find a security issue in Focusmate Companion, please report it by email:

```txt
ikramsalim.tech@gmail.com
```

Please include:

- A short description of the issue
- Steps to reproduce it
- Browser and operating system
- Any screenshots or logs that help explain the issue

## Security model

Focusmate Companion is designed to keep the extension small and local-first.

- No backend server
- No analytics
- No external data transmission
- Local browser storage only
- Limited host permission for `https://app.focusmate.com/*`
- No Chrome `tabs` permission
