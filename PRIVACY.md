# Privacy Policy

Focusmate Companion does not collect, sell, share, or transmit personal data.

## Data storage

The extension stores settings and task lists locally in the browser using `chrome.storage.local` or `browser.storage.local`.

Stored data may include:

- Audio cue preferences
- Volume settings
- Ambient sound preferences
- Task list items
- Completed task status

## Data transmission

Focusmate Companion does not send user data to any external server.

The extension does not use:

- Analytics
- Advertising trackers
- Remote logging
- Third-party data sharing
- External user accounts

## Site access

The extension runs only on:

```txt
https://app.focusmate.com/*
```

This access is used to detect Focusmate session timing and play local audio cues.
